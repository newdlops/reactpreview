# 아이콘 자산

`icon.png`는 React File Preview의 Visual Studio Marketplace 아이콘입니다.

- 형식: PNG
- 크기: 256×256
- 배경색: `#12182F`
- 용도: `package.json.icon`과 Marketplace gallery
- 개념: 코드 pane에서 시각적 preview pane으로 이어지는 서버 없는 파일 프리뷰

2026-07-15에 OpenAI 내장 이미지 생성 도구로 이 프로젝트만을 위해 생성하고 256px로
다운스케일했습니다. React atom이나 Visual Studio Code 로고를 복제하지 않은 원본 심볼이며,
Marketplace 배포 시 `assets/icon.png`만 VSIX에 포함하고 이 설명 파일은 제외합니다.

최종 생성 지시의 핵심은 다음과 같습니다.

> Original square developer-tool icon combining two overlapping panes, code angle brackets and a
> preview eye; midnight navy, cyan, coral and soft white; strong silhouette; no text, trademarks,
> React atom, Visual Studio Code logo, watermark or tiny details.
